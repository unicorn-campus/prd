# 솔루션 탐색

## 킹핀 문제 및 방향성
- 킹핀 문제: 결제 순간, 정보가 흩어져 있어 실시간 확인 실패
- Needs Statement: 포인트와 혜택을 매번 찾아다니는 게 귀찮은 바쁜 실속형 직장인은, 여러 화면을 일일이
  뒤지지 않고도 손해 보지 않기 위하여, 지금 나에게 필요한 정보를 내가 찾지 않아도 앱이 먼저 알아서
  모아 보여주는 경험이 필요함

## 타 산업 사례 활용 요약 (Steal & Synthesize, 5개 이상)
- 금융 슈퍼앱(토스뱅크/토스): 자산 통합 화면, 자동 저축·자동 할인
- 카드업계(신한플레이, PLCC 위젯, 삼성페이): 즉시 거래통지, 근접 자동 인식, 위젯 상시 노출
- 이커머스(장바구니 리마인드, 배달앱 마감임박 배지): 소멸 방지 리마인드, 긴급성 표시
- 스타벅스 사이렌오더/딥브루: 매장 근접 자동 활성화, 맥락 기반 추천
- 은행권 오픈뱅킹/마이데이터: 원장 통합 조회 표준
- 넷플릭스 예측 프리캐싱: 결제 전 혜택 정보 사전 반영
- 항공업계(마일리지 가족합산, Wallet 탑승권): 포인트 이전, 위젯화
- 보안업계 버그바운티: 고객 참여형 검증
